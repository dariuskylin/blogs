经常要将手机拍摄的照片上传，但照片一般都比较大，所以每次都要修改一下大小，机器里只装有Photoshop，确实不愿意打开它来做这件事，于是就萌发了写一个简单方便修改图片大小的程序。Python非常适合做这件事。

ResizeImage

运行截图：

介绍：

ResizeImage可以创建图片的缩略图，该程序需要输入3到4个参数（每个参数用空格进行分隔），它们的意思分别是： 

1、图片目录或是图片的路径
2、需生成图片的宽，整数
3、需生成图片的高，整数
4、如果第1个参数是图片目录，可以使用这么4个参数来过滤图片的类型，比如，只想对目录下所有的gif图片进行缩略图的创建。该参数包括jpg,jpeg,gif和png。如果不指定它，默认的值为jpg。意为着，程序会将该目录下所有格式为jpg的图片进行缩略图的创建。

使用方法： 

1、运行resizeImage.exe
2、将图片或图片目录拖到程序界面（命令行）里。当然也可以手动输入图片或图片目录所在路径
3、输入需要生成图片的宽高

示例： 

1、将图片目录d:\pics下所有格式为jpg的图片，创建宽为300或高为300的缩略图
Enter path width height: d:\pics 300 300

2、将图片目录d:\pics下所有格式为gif的图片，创建宽为300或高为300的缩略图
Enter path width height: d:\pics 300 300 gif

3、将图片d:\pics\ex.jpg，创建宽为300或高为300的缩略图
Enter path width height: d:\pics\ex.jpg 300 300

PS：创建的缩略图为什么是宽为300或是高为300，（300这个数值由您输入的决定，这里只是引用上面例子中的数值）

因为程序会根据图片比例进行自动的等比例缩放，所以，可能是宽被控制为300，也可能是高被控制为300

技巧，关于缩略时的宽高取舍

程序在创建缩略图的时候，永远是按等比例来进行，这里就涉及到创建缩略图的时候，是取目标宽为基准，还是取目标高为基准。

程序的做法是：永远以小值为基准。如：输入宽高分别为：300 500，那么创建的缩略图将是300宽为基准，进行等比例缩略。如果输入宽高分别为：500 300，那么创建的缩略图就将是以300高为基准。如果宽高相等，500 500，那么将根据图片实际的宽高，以长的一边为基准，如：原始图片宽高为2592*1936，若输入500 500，那创建缩略图大小为：500*373，反之同样成立
