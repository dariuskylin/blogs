    在github搭建blog之前，当然需要有一个github账户；本机也要安装好git。

根目录文件结构

    选择一个空目录或者新建一个目录，用来作为主目录primary_dir，进入primary_dir执行 {% highlight php %} git init {% endhighlight %}

基本的文件结构应该是：

/primary_dir
                |-- _config.yml  //配置文件
                |-- _layouts     //模板目录
                |-- _posts       //日志文件目录
                |-- index.html   //首页文件
